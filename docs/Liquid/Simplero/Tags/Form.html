Overview

Renders an HTML <form> for an object. Only certain types of objects can be used with this.

{% form 'comment' %}                  ... {% endform %}
{% form 'comment' to comment %}       ... {% endform %}
{% form 'comment_edit' for comment %} ... {% endform %}
