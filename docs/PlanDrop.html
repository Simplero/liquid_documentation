#contact_cost_data ⇒ Object

Returns slabs of contacts and their cost. Also, increase in costs for each step increase in contacts

Example:

{
  thresholds: {
    1000  => 0,
    2500  => 20,
    5000  => 50,
    7500  => 70,
    10000 => 90
  },
  step:       2500,
  cost:       15
}
